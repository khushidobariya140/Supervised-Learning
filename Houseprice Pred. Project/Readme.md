# Advanced Regression House Price Prediction

## Project Overview

This project focuses on predicting house prices using Machine Learning Regression algorithms. The dataset contains various features related to houses such as area, number of bedrooms, bathrooms, floors, parking spaces, furnishing status, and other property-related information.

The main objective is to build regression models that can accurately predict house prices based on the available features.

---

## Problem Statement

House price prediction is an important real-world application of Machine Learning. The goal of this project is to analyze housing data, perform preprocessing and feature engineering, and develop regression models to estimate house prices.

---

## Dataset Information

- Dataset Name: Advanced Regression House Price Dataset
- Total Records: 3800
- Target Variable: Price
- Dataset Type: Regression

## Project Workflow

### 1. Data Collection
- Load dataset using Pandas.
- Understand dataset structure.

### 2. Data Understanding
- View dataset shape.
- Check data types.
- Generate statistical summary.

### 3. Data Cleaning
- Check missing values.
- Handle missing values if present.
- Check duplicate records.

### 4. Exploratory Data Analysis (EDA)
- Distribution of target variable.
- Correlation heatmap.
- Histograms of numerical features.
- Boxplots for outlier detection.

### 5. Feature Engineering
- Create additional features from existing columns.
- Convert date-related columns if available.
- Encode categorical variables.

### 6. Data Preprocessing
- Feature selection.
- Train-Test Split.
- Feature Scaling.

### 7. Model Building

#### Linear Regression
Used as the baseline regression model.

#### Random Forest Regressor
An ensemble learning method that improves prediction accuracy.

#### Support Vector Regression (SVR)
A machine learning algorithm that predicts continuous values using support vectors.

### 8. Model Evaluation

The following metrics are used:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

### 9. Model Comparison
Compare the performance of all regression models and identify the best-performing model.

## Evaluation Metrics

### Mean Absolute Error (MAE)
Measures the average prediction error.

### Root Mean Squared Error (RMSE)
Measures the square root of average squared prediction errors.

### R² Score
Measures how well the model explains the variance in the target variable.
